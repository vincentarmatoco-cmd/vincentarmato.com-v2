# Best Game-Day photos

These 8 photos feed the **Featured → Game Days** row at the top of the
homepage (`index.html`, the `.featured` section). This is your curated "best
work" — the first photography a visitor sees.

- Files are named `best-01.jpg` … `best-08.jpg` and display in that order.
- They're seeded with a spread from the game galleries — **swap any by
  replacing the file** with the same name (`best-03.jpg`, etc.).
- Landscape/horizontal crops look best (these tiles are 3:2 for game action).
- ~1200px wide is plenty. Drop full-size photos here and I'll optimize them.

To change how many show, edit the `.featured-grid` for Game Days in
`index.html`.
